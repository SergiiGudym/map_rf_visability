# RF Visibility Web (Kotlin Multiplatform + Compose)

MVP веб-додаток для аналізу радіо-видимості на базі локального `output_hh.tif` + OpenStreetMap.

## Що реалізовано

- Kotlin Multiplatform (JS browser target) + Compose Web UI.
- Ліва панель налаштувань + велика права частина з картою.
- Завантаження локального `*.tif` / `*.tiff` (GeoTIFF) через file input.
- Підключення OpenStreetMap через Leaflet.
- Контур області GeoTIFF (bbox) поверх OSM.
- Клік всередині контуру задає центр.
- Параметри: радіус, висота обладнання (центр).
- Рендер карти покриття (спрощено, без Френеля):
  - до 5м — прозоро,
  - до 50м — зелений,
  - до 200м — синій,
  - до 500м — червоний,
  - 1500м+ — чорний.
- Ключові пороги і прозорість у лівій панелі редагуються.

## Примітки по моделі

`output_hh` інтерпретується як значення необхідної висоти над землею в точці.
Покриття розраховується як `max(0, output_hh - height_center_equipment)` у радіусі від центру.

## Запуск

```bash
gradle jsBrowserDevelopmentRun
```

> Якщо у вашому середовищі заблоковано доступ до Gradle plugin portal/Maven, спочатку потрібно увімкнути мережевий доступ або підготувати локальний кеш залежностей.
